## ✅ コードで実施した改修（このブランチに含まれています）

### 1. 構造化データ（JSON-LD / Person スキーマ)を追加 — SEO
**ファイル:** `src/pages/masterPage.js`(全ページ共通で読み込まれる)

検索エンジンに対して「林宋其 = 人物(Person)」であることを、機械が読める形(JSON-LD)で伝えます。
氏名・別名(Soki Hayashi)・肩書き・出身大学・専門領域・公開URLを構造化して渡します。

- **効果:** Googleが人物として正しく認識し、ナレッジパネルやリッチな検索結果の土台になる。
- **見た目への影響:** ゼロ(裏側の情報のみ)。
- **安全策:** 万一APIが失敗してもサイト表示には影響しないよう `try/catch` で囲んでいます。

> 🔸 **林さんに1つお願い(任意):**
> `sameAs` の欄が**空**です。ここに YouTube「アートゥーン!／Artoone!」や各SNSの
> **正式なURL** を入れると、それらが「同一人物の別アカウント」として検索エンジンに伝わります。
> 私が勝手に推測URLを入れると誤リンクのリスクがあるため、空のままにしています。
> URLを教えていただければ追記します。

> 🔸 **代替案:** 同じ内容を Wixエディタの「SEO設定 → 詳細設定(Advanced)→ 構造化データ」に
> 貼る方法もあります(そちらはサーバー側で出力されるため、よりクローラーに優しい)。
> その場合はコード側を消して二重化を避けます。どちらにするか相談可。

### 2. アクセシビリティ / 描画のグローバルCSS — UIUX
**ファイル:** `src/styles/global.css`(これまで空でした)

特定の要素IDに依存しない、サイト全体に安全に効く標準的な改善のみ:

| 内容 | 何のため |
|---|---|
| 文字のなめらか描画(font-smoothing) | 高精細ディスプレイで文字を綺麗に |
| キーボード操作時のフォーカス表示(`:focus-visible`) | アクセシビリティ。マウス操作時には出ないのでデザインの邪魔をしない |
| `prefers-reduced-motion` 対応 | 「動きを減らす」設定の人に配慮(標準的なa11y対応) |

- **効果:** アクセシビリティ向上 → 検索評価のUXシグナルにもプラス。
- **要確認:** 反映は `wix dev`(Local Editor)で目視確認してください。
  もしWix側で Custom CSS が無効だと適用されない可能性があります(その場合も害はありません)。

## ⚠️ コードでは触れない=Local Editor / エディタ専用の範囲

以下は **Wixのビジュアルエディタ専用** で、コードからは実施できません。林さんの作業になります。

### レスポンシブ(スマホ/タブレット表示)
- ブレークポイントごとのレイアウト・文字サイズ・余白・要素の表示/非表示は、
  Wixエディタ上部の **デバイス切り替え(PC / タブレット / モバイル)** で1つずつ調整します。
- これは「目で見て動かす」作業のため、コード化できません。
- **おすすめ確認手順:** エディタでモバイル表示に切り替え、(1)文字のはみ出し (2)画像の潰れ
  (3)タップしにくい小さなボタン (4)横スクロールの発生、の4点をチェック。

### ページ単位のSEO設定(タイトル/meta description)
- 各ページの **SEO設定** で入力。→ 詳細は `docs/wix-priority1-checklist.md` のタスク4参照。

### 画像まわり(SEO/表示速度に直結)
- 各画像の **代替テキスト(alt)** はエディタで設定します(SEO・アクセシビリティ両方に効く)。
- 特にWorksの20点以上のサムネイルにalt未設定が多いと、検索でもアクセシビリティでも損。
- 画像の圧縮・遅延読み込み(lazy load)はWixが概ね自動でやりますが、
  **極端に大きい元画像** はアップロード前に適切なサイズへ。

## 公開について(重要)

これらのコード変更は **git push でWixに同期されますが、自動公開はされません。**
本番サイトに反映するには、林さんが確認のうえで **`wix publish`**(またはエディタの「公開」)を実行してください。

```bash
git pull origin <ブランチ名>   # 最新を取得
wix dev                        # Local Editorで見た目を確認
# 問題なければ
wix publish                    # 本番反映
```
