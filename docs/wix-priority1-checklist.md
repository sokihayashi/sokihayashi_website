# 優先度1:緊急修正チェックリスト(Wix Local Editor 作業)

> 📌 **このドキュメントについて**
> 優先度1の修正5点は、すべて **Wixのビジュアルエディタ側** の作業です(コードでは触れません)。
> Local Editor を起動して、このチェックリストを見ながら上から順に進めてください。
> 各タスクに「貼り付ける確定テキスト」または「要確認のたたき台」を用意しています。

---

## 0. はじめる前に(毎回の習慣)

ターミナルでこのフォルダに入って、以下を実行します。

```bash
git status          # いまの状態を確認(編集前)
wix dev             # Local Editor をブラウザで起動
```

- `wix dev` を実行すると、ブラウザでWixの編集画面が開きます。ここで文字や画像を直接編集できます。
- 文章・画像・URL・SEO設定は、すべてこの **Local Editor 側** で編集します。
- 編集が終わったら、画面右上の **「保存」** を押すと変更がWixに同期されます。

> ⚠️ 注意:Wixのエディタは、サイトのテンプレート世代によってメニューの名前や場所が少しずつ違います。
> 下記の手順でボタン名が見つからない場合は、近い名前のメニューを探すか、林さんから私に「この画面でどこを押せばいい?」と聞いてください。

## ✅ タスク5:ページURL名の変更

**場所:** 各ページの設定 → URLスラッグ
**状態:** 確定 — 変更内容は明確

| ページ | 現在のURL | 変更後 |
|---|---|---|
| Profile | `/blank-2` | `/profile` |
| Contact | `/blank-3` | `/contact` |

### 操作場所の見つけ方
1. Local Editor 左側の **「ページ」メニュー** を開く。
2. 対象ページの「**…**」→ **「設定」** をクリック。
3. **「ページ情報(Page Info)」** タブの **「URLスラッグ(URL slug)」** 欄を書き換える。
   - Profileページ:`blank-2` → `profile`
   - Contactページ:`blank-3` → `contact`

> ⚠️ **URL変更時の注意:** すでに古いURL(`/blank-2` 等)を誰かに共有している場合、変更すると旧リンクは404になります。
> Wixには「URLリダイレクト(301)」を設定する機能があるので、心配なら旧URL→新URLの転送設定もしておくと安全です(設定場所が分からなければ私に聞いてください)。

## メモ:この5点が「コードで触れない」理由

Wixのサイトは2層構造です。

- **ビジュアルエディタ層**(Wixサーバー上) … 文字・画像・レイアウト・URL・SEO。**今回の優先度1は全部こちら。**
- **コード層**(このGitリポジトリ) … ボタンの動きやデータ読み込みなどの「仕掛け」だけ。

そのため、優先度1の修正はGitのコードには現れず、Local Editor での作業になります。
